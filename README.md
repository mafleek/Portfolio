# maleekpatterson.com (static GitHub Pages)

This is a zero-build, static portfolio site (HTML/CSS/JS) designed around an editorial-luxury aesthetic (blood-rose accent + glass).

## What’s already wired
- Home page + Work grid
- Case studies:
  - HEART Clinic (prototype + downloads)
  - SpinSpotter (prototype + downloads + Canva deck link)
- About page (headshot included)
- Resume page (embed + download)

## What you need to update
1) **Email link**
- In `index.html` and `about.html`, search for `mailto:your-email@domain.com` and replace with your real email.

2) **Resume PDF**
- Put your resume at: `assets/downloads/Resume.pdf`

3) Optional: add more screenshots
- Add images to `assets/img/` and link them in the project pages.

## Deploy on GitHub Pages
1) Create a repo (example: `maleekpatterson-com`)
2) Upload everything in this folder (keep file structure)
3) GitHub:
   - Settings → Pages
   - Source: `Deploy from a branch`
   - Branch: `main` / folder: `/root`
4) Set custom domain to `maleekpatterson.com`
